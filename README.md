🚀 TECHFEST 2026 — Neural Void

An immersive futuristic event landing page built with HTML, CSS, JavaScript, and Three.js, featuring a cinematic 3D environment, scroll-driven camera navigation, interactive event tracks, speaker showcases, and registration experience.

✨ Overview

TECHFEST 2026 — Neural Void is a sci-fi inspired conference website that combines advanced front-end animation techniques with a fully immersive user experience.

The website simulates a futuristic digital environment where visitors can:

Explore event tracks
Discover keynote speakers
Navigate through a 3D world
Register for the conference
View live countdowns and event schedules

Everything is contained within a single self-contained HTML file.
